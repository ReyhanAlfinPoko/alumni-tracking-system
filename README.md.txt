# Alumni Tracking System

## Deskripsi
Aplikasi web sederhana untuk membantu administrator kampus melacak data alumni.

## Fitur
- Menambah data alumni
- Menampilkan daftar alumni
- Pencarian alumni

## Teknologi
HTML, CSS, JavaScript

## Pengujian Aplikasi

| No | Fitur | Aspek Kualitas | Cara Uji | Hasil |
|----|------|---------------|---------|------|
|1|Tambah Alumni|Functionality|Input data alumni|Berhasil|
|2|Menampilkan Alumni|Usability|Data muncul di list|Berhasil|
|3|Pencarian Alumni|Performance|Cari nama alumni|Cepat|
|4|Tampilan UI|Usability|User mencoba sistem|Mudah digunakan|
